# OW2 승패분석 프로젝트
오버워치는 2로 접어든 이후 오랫동안 유저 게임결과 데이터를 공개하지 않았다.  
최근 유저 데이터를 다시 공개하였으나, 나 포함 대부분의 유저가 '프로필 비공개'를 설정하고 있기 때문에 여전히 게임 결과를 확인하기는 쉽지 않다.  
이에 적어도 나의 게임 결과만이라도 직접 기록해서 분석하자는 취지로 시작한 프로젝트다.  

데이터 기록은 25년 1월에 시작했다.  
오버워치는 출시 당시부터 꾸준히 했었지만, 경쟁전은 돌리지 않고 일반전만 돌렸다.  
이 때 드디어 경쟁전을 제대로 시작할 맘을 먹었는데, 이참에 승패기록을 처음부터 직접 수집하고 싶어서 기록을 시작했다.  

그렇게 2개월 정도 데이터를 모으다가 25년 3월경 분석을 시작했고, 소스코드를 지속적으로 개량해 오고 있다.  

environment: python3.13(pyenv), poetry 기반 pandas, plotly 사용

## 원본 데이터
직접 생성.  
'오버워치 승패 기록표.xlsx' image 이런 식으로 작성하였다.  
2025년 9월 12일 현재 577개의 데이터가 있다.

## 분석 실행코드
generate_report.py를 실행한다.

## 분석 소스코드
2025년 9월 12일 현재 'source_generate_report_20250822.py' 파일을 사용 중이다.  
다음에 따라 분류된 게임 기록별로 승패합계와 승률 그래프를 그리는 코드다.  
* 경쟁전 전체
* 경쟁전 최근 100판
* 일반전 전체
* 일반전 최근 100판
* 경쟁전+일반전 전체
* 경쟁전+일반전 최근 100판

## 지표 설명
* 승패합계: 게임별로 이기면 +1, 지면 -1, 무승부면 0을 누계했음.  
* 승률: (승리한 게임 수) / (전체 게임 수)

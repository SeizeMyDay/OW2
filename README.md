# OW2 승패분석 프로젝트
25년 3월경 시작했다. 당시 연패를 반복하고 있었는데, 내 장기간의 승패기록을 보고 싶었지만 당시 오버워치는 유저 게임결과 데이터를 공개하지 않았다.  
그래서 직접 승패기록을 작성하여 내 연패에 어떤 이유가 있는지 분석하고자 본 프로젝트를 시작하게 되었다.

environment: python3.13(pyenv), poetry 기반 pandas, plotly 사용

## 원본 데이터
직접 생성.  
'오버워치 승패 기록표.xlsx' image 이런 식으로 작성하였다.  
2025년 9월 12일 현재 577개의 데이터가 있다.

## 분석 실행코드
generate_report.py를 실행한다.

## 분석 소스코드
2025년 9월 12일 현재 'source_generate_report_20250822.py' 파일을 사용 중이다.  
경쟁전 전체, 경쟁전 최근 100판, 일반전 전체, 일반전 최근 100판, 경쟁전+일반전 전체, 경쟁전+일반전 최근 100판별로 각각의 승패합계와 승률 그래프를 그리는 코드다.

## 지표 설명
* 승패합계: 게임별로 이기면 +1, 지면 -1, 무승부면 0을 누계했음.  
* 승률: (승리한 게임 수) / (전체 게임 수)
